# My AI Fluency Workbook

This is your personal workbook for the **Become an AI-Fluent Agile Coach** program.

You created this from the [workbook template](https://github.com/ampyard/ai-fluent-agile-coach-workbook). The course instructions live in the [course repo](https://github.com/ampyard/ai-fluent-agile-coach) — this is where your work goes.

---

## How to use this repo

- **Save prompts** → `artifact-library/` (use the template in `templates/prompt-template.md`)
- **Save bots** → `artifact-library/bots/`
- **Save workflows** → `artifact-library/automation/` 
- **Save tools** → `tools/` (one `.html` file per tool)
- **Track progress** → `progress.md`
- **Reflect weekly** → `week-XX-name/reflection.md` or a single running file

Each week folder has a `notes.md` for jotting anything mid-week.
